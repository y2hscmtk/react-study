---

명령어 정리

```bash
# Vite 최신 버전 설치
npm install vite@latest

# 새로운 Vite 리액트 프로젝트 생성 (my-app 부분은 원하는 폴더명으로 변경)
npm create vite@latest my-app -- --template react

# 프로젝트 의존성 설치
npm install

# 개발 서버 실행 (로컬에서 앱 미리보기)
npm run dev

# 빌드(프로덕션용 번들 생성)
npm run build

# 빌드 결과물 미리보기(로컬 서버)
npm run preview
```
